# MathAdventureQuest

Prepare-se para uma jornada emocionante pelos fundamentos da matemática!  
Embarque nesta aventura educacional em **2D**, criada com a poderosa engine **Godot**. Desafios matemáticos estimulantes aguardam você em um mundo interativo e cativante.

## 📚 Sobre o Projeto

**MathAdventureQuest** é um jogo educacional desenvolvido com o objetivo de **ensinar matemática para crianças de forma divertida, dinâmica e interativa**.

O jogo utiliza **mecânicas de puzzle** para transformar conceitos matemáticos em desafios que estimulam o **raciocínio lógico**, a **resolução de problemas** e o **aprendizado progressivo**. Ao longo da jornada, os jogadores exploram diferentes desafios matemáticos em um ambiente lúdico, incentivando o aprendizado enquanto jogam.

A proposta é tornar o ensino da matemática mais **engajador e acessível**, utilizando elementos de jogos para criar uma experiência educativa envolvente.

## 🎮 Principais Características

- 🧩 **Puzzles matemáticos interativos**
- 🎯 Aprendizado de matemática de forma **lúdica**
- 📈 **Progressão de dificuldade** para reforçar conceitos
- 🧠 Estímulo ao **raciocínio lógico**
- 🌎 Ambiente 2D interativo e educativo

## 🛠️ Tecnologias Utilizadas

- **Godot Engine** – Desenvolvimento do jogo
- **GDScript** – Linguagem de programação utilizada na lógica do jogo
